# Unsplash Search

## Get Started

1. Create `.env` file with following content:

```
REACT_APP_UNSPLASH_ACCESS_TOKEN=your_access_token
REACT_APP_UNSPLASH_SECRET_KEY=your_secret_token
```
2. Run `yarn`.
3. Run `yarn start`
